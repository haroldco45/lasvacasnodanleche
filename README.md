# 🐄 Las Vacas No Dan Leche — App Educativa PWA

> *"Hay que ordeñarlas. Nada es gratis, pueblo. El que trabaja, elige. El que no trabaja, obedece."*

## Descripción

Aplicación web progresiva (PWA) interactiva, animada y educativa que enseña una de las lecciones más importantes de la vida a través de la metáfora de la vaca lechera.

**Mensaje central:** Las vacas no *dan* leche espontáneamente — hay que *ordeñarlas*. Todo resultado en la vida requiere acción, esfuerzo y trabajo. Lo regalado se acaba y sale caro.

## Características

- 🐄 **Vaca interactiva animada** — Toca/haz clic para ordeñar
- 🥛 **Cubeta con nivel de leche** que sube con cada ordeño
- 🏆 **Sistema de logros gamificado** — 8 niveles de progreso
- 📚 **6 lecciones educativas** con animaciones de scroll
- 💬 **12 frases filosóficas** rotativas sobre el trabajo y la vida
- 🎊 **Celebraciones con confeti** en hitos especiales (1, 5, 15, 30, 50 litros)
- 💾 **Persistencia local** — tu progreso se guarda entre sesiones
- 🌄 **Escena animada** — cielo, sol, nubes, estrellas, pasto
- 📱 **Instalable como app** (Android automático, iOS con instrucciones)
- ⚡ **Service Worker** para uso offline

## Lecciones incluidas

1. **La verdad sobre las vacas** — Nada llega sin acción
2. **Lo regalado sale caro** — El costo forma el carácter
3. **Lo regalado se acaba** — Invierte en capacidades, no en dádivas
4. **Elige con inteligencia** — El voto también requiere trabajo intelectual
5. **El esfuerzo es el camino** — Dignidad, independencia y orgullo
6. **Siembra hoy, cosecha mañana** — Colombia tiene todo; falta la mentalidad

## Tecnología

- HTML5 + CSS3 + JavaScript vanilla
- PWA con Web Manifest dinámico
- Service Worker para caché offline
- localStorage para persistencia de progreso
- SVG animado para la vaca
- IntersectionObserver para animaciones de scroll
- CSS animations y keyframes

## Instalación / Despliegue

### GitHub Pages
```bash
git clone https://github.com/haroldco45/vaca-no-da-leche
cd vaca-no-da-leche
# Sube index.html a la raíz del repositorio
# Activa GitHub Pages en Settings → Pages → main branch
```

### cPanel / hosting
Sube `index.html` a la carpeta deseada. No requiere backend ni base de datos.

## Uso

Simplemente abre el archivo en cualquier navegador moderno. Para instalarlo como app:
- **Android/Chrome:** Aparecerá banner automático "Instalar app"
- **iPhone/Safari:** Compartir → "Agregar a pantalla de inicio"
